# 日本史探究 No.11-22 学習アプリ

授業プリント「日本史探究 No.11-22」を見ながら、要点確認・暗記カード・問題演習で学習できるPWAです。

## iPhoneで使う

1. GitHub PagesのURLをSafariで開く。
2. 共有ボタンを押す。
3. 「ホーム画面に追加」を選ぶ。
4. 追加したアイコンから起動する。

ホーム画面から起動すると、PWAとして単独画面で表示されます。初回表示後は主要ファイルがキャッシュされます。

## ローカル確認

```powershell
python -m http.server 5173
```

ブラウザで `http://127.0.0.1:5173/` を開きます。
